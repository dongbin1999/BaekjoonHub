### 입력 

 <p>첫째 줄에 테스트 케이스의 수 $T$가 주어진다.</p>

<p>이후 $T$개의 줄에 걸쳐 $N$이 주어진다.</p>

### 출력 

 <p>각 테스트 케이스마다 만들 수 있는 서로 다른 칼날의 개수를 $1\,000\,000\,007$로 나눈 나머지를 출력하라.</p>
